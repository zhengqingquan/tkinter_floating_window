# tkinter_floating_window
A Tkinter window that will not be killed by the display desktop

这是一个tkinter的悬浮窗口，并且不会被win10的显示桌面给隐藏掉。甚至还可以自由拖动窗口位置。
但缺点是无法将窗口置为顶层（用于显示在最前面）。或许在之后可以找到方法修改。
